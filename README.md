# chatmessage

Real-time messenger web app built with **MongoDB, Express, React, Node.js, and Socket.io**.

## Requirements

- Node.js 20+
- MongoDB running locally or remotely

## Setup

```bash
cd /home/runner/work/chatmessage/chatmessage
npm install
```

Create `.env` file:

```env
MONGODB_URI=mongodb://127.0.0.1:27017/chatmessage
PORT=3000
```

## Run

```bash
npm run dev
```

Open `http://localhost:3000`.
